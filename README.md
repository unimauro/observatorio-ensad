# Observatorio ENSAD

Portal ciudadano de transparencia de la **Escuela Nacional Superior de Arte Dramático "Guillermo Ugarte Chamorro"** (ENSAD), Unidad Ejecutora 123 del MINEDU. Solo datos públicos.

**Live:** https://unimauro.github.io/observatorio-ensad/

Módulos: presupuesto y ejecución (MEF/SIAF UE 123), en qué se gasta cada sol, proveedores y dueños (OECE/CONOSCE), personal (Portal de Transparencia). Nunca nombres de estudiantes.

Hermano de [Observatorio UNI](https://unimauro.github.io/observatorio-uni/). No es un portal oficial de la ENSAD. Correcciones: carlos@cardenas.pe
